# Mohammed Rinshad P - Data Analyst Portfolio

Welcome to the personal portfolio of Mohammed Rinshad P, a Data Analyst proficient in SQL, Python, Power BI, and Tableau.

## Overview
This repository contains the source code for my interactive portfolio, showcasing my skills, experience, and projects in the field of Data Analytics. It emphasizes clean aesthetics and dynamic transitions to deliver an engaging user experience.

## Features
- **Profile & Hero Sections**: Introduces my background and core competencies.
- **Skills Showcase**: Visual representation of both technical and soft skills.
- **Responsive Design**: Ensuring a seamless experience across desktop and mobile devices.
- **Modern UI**: Utilizing dark themes, deep blue accents, and CSS-based micro-animations for an elegant look.

## Technologies Used
- HTML5
- CSS3 (Vanilla)
- JavaScript

## Running Locally
To run this project locally, you can use any static file server, or run the following if you have Node.js installed:

```bash
npm run dev
```
